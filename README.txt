This is a description of the four categories of the file.

Data: Contains the data used in the analysis in the original form prior to cleaning. Recommended to have excel to see the data outright, though it can also be seen in R studio. Includes brief description of the data.

Presentation: Contains the presentation used when the project results were presented to the class. Recommened to have powerpoint to access.

Project: Contains the RMD and HTML files of the semester long project, including visualizations of the data, summaries, and analysis of it. Recommened to have R studio to access the RMD file.

Proposal: The Proposal for the analysis suggested, fixed to account for use of R studio over Excel.
